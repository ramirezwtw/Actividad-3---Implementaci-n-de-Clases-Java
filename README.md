# Actividad: Implementación de Clases en Java

Este proyecto es una demostración de los conceptos fundamentales de POO (Programación Orientada a Objetos) en Java, incluyendo constructores, encapsulamiento y sobreescritura de métodos.

## Contenido
- `Book.java`: Clase principal con atributos privados, getters, setters y validaciones.
- `Main.java`: Clase de prueba para demostrar el funcionamiento de la lógica.

## Instrucciones de Ejecución
1. Abre una terminal en la carpeta del proyecto.
2. Compila las clases:
   ```bash
   javac *.java
